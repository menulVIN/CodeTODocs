Packaging instructions

Run `package_exe.bat` to create `CodeDocs_Shared.zip` in the workspace root. This ZIP will contain the built `CodeDocs.exe` you pointed out.

After that, you can:
- Upload the ZIP to a file share (Dropbox, Google Drive, WeTransfer) and share the link.
- Attach the ZIP to a GitHub Release on your repository for easy distribution.
- If you prefer, I can create an installer script (Inno Setup) — it's already added as `installer.iss` — and advise on signing and release notes.
